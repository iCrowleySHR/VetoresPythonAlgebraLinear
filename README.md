# Exercícios de Vetores em Python

Este repositório contém **5 exercícios simples de vetores usando NumPy**.

## Exercícios

1. **Soma de vetores**
   Somar dois vetores.

2. **Subtração de vetores**
   Calcular a diferença entre dois vetores.

3. **Produto escalar**
   Calcular o produto escalar entre dois vetores.

4. **Norma de um vetor**
   Calcular o tamanho (norma) de um vetor.

5. **Combinação linear**
   Encontrar valores `a` e `b` tais que:

   ```
   w = a*u + b*v
   ```

## Tecnologias

* Python
* NumPy

## Como executar

Instale o NumPy:

```
pip install numpy
```

Execute os arquivos Python com:

```
python nome_do_arquivo.py
```
